# Series:

Returns info about the series a book is part of

### URL:

`https://api.wattpad.com/reading/v1/stories/{story_id}:seriesInfo`

### Fields:

This endpoint returns a [Series](../Data_Types/Series.md).

### Error Codes:

None

### Example Usage:

`https://api.wattpad.com/reading/v1/stories/242618522:seriesInfo`
