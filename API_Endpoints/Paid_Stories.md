# Paid Stories:

Return a list of paid stories.

### URL:

`https://www.wattpad.com/v4/paid_stories/`

### Fields:

| Field | Data Type |
| - | - |
| stories | List ([Story](../Data_Types/Story.md)) |
| nextUrl | URL |

### Example Usage:

`https://www.wattpad.com/v4/paid_stories/`

`https://www.wattpad.com/v4/paid_stories/?fields=stories(id,title)`
